# Manual QA Portfolio

Learning portfolio project for practicing TypeScript, Jest, and basic automated testing.

## Project Goal

This repository is part of my QA learning path. It contains simple TypeScript utility functions and unit tests that verify expected behavior.

## What Is Inside

- `src/utils.ts` - math utility functions: add, subtract, multiply, and divide.
- `tests/example.test.ts` - Jest unit tests for the utility functions.
- `jest.config.js` - Jest configuration for TypeScript tests.
- `tsconfig.json` - TypeScript configuration.

## Installation

```bash
npm install
```

## Available Commands

Run tests:

```bash
npm test
```

Build the project:

```bash
npm run build
```

Run the compiled app:

```bash
npm start
```

Watch tests while learning:

```bash
npm run test:watch
```

## Test Coverage

Current tests check:

- addition
- subtraction
- multiplication
- division
- division by zero error handling

This is a learning project and will be expanded with more QA practice examples.
